# Backend Query Layer

当前目录先实现一个不依赖 Web 框架的查询层。

原因很简单：

- 当前本地环境还没有 `FastAPI` 和 `uvicorn`
- 但查询逻辑已经可以基于本地 `SQLite` 数据仓库先跑通
- 后续无论接 `FastAPI`、`Flask` 还是其他后端框架，都可以复用这层

当前文件：

- `query_service.py`

当前支持的能力：

- 大盘概览汇总查询
- 平台营收占比
- 平台订单量对比
- 平台客单价对比
- 趋势查询
- 门店 Top20 查询
- 城市 Top20 查询

当前限制：

- 这还不是正式 HTTP API
- 它是后端数据查询层
- 等本地或正式环境装好 `FastAPI` 后，再把这层包装成接口即可
